Clock Web App

Thank you for visiting the Clock Web App! There are 4 features you can try out!

This feature displays the current date and time.

This feature is a digital stopwatch that counts up time; you can pause while the stopwatch is active, and resume to unpause, as well as reset the stopwatch time.

This feature allows you to create an alarm for a specific time (the day of) and when the alarm time set matches the current time, it will beep continuously until you shut the alarm off via "Reset Alarm" or via the notification that will appear on the bottom for a short period of time.

This feature allows you to set a timer, and when the timer hits zero, it will continuously beep until you shut the timer off via "Reset" or via the notification that will appear on the bottom for a short period of time.
